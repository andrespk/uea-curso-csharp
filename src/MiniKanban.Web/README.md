# MiniKanban Web

Frontend em Vite + React + TypeScript + TailwindCSS v4 para o MiniKanban.

## Setup

```bash
npm install
npm run dev
```

Acesse: `http://localhost:5173`

## Build

```bash
npm run build
npm run preview
```

## Features

- ✨ Login com JWT
- 📊 Dashboard com lista de boards
- 🎯 Kanban board com drag & drop
- 🌙 Dark mode premium com glassmorphism
- ⚡ TailwindCSS v4 com animações suaves
- 📱 Responsivo
